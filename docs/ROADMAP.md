# Roadmap

이 문서는 트레이딩봇 프로젝트의 향후 개발 방향을 정리합니다.

README에는 현재 실제 구현된 기능과 사용법만 작성하고, 아직 구현되지 않은 기능은 이 문서에서 관리합니다.

---

## 문서 관리 원칙

```txt
README.md = 현재 실제 동작하는 기능과 사용법
docs/ROADMAP.md = 앞으로 만들 기능과 방향성
CHANGELOG.md = 지금까지 변경된 내용
```

---

## v0.x - 국내장 실전 자동매매 안정화

국내장(KRX) 실전 자동매매를 안정화하는 단계입니다.

* 국내장(KRX) 실전 자동매매 안정화
* 매수/매도 조건 정리
* 주문 처리 안정화
* 체결 조회 안정화
* 잔고 조회 안정화
* 거래 로그 기록 고도화
* 일일 매매 리포트 개선
* 매수될 뻔한 종목 분석 기능 개선
* 리스크 관리 규칙 강화
* 일일 손실 제한 관리
* 동시 보유 종목 수 제한 관리

---

## v1.x - 데이터 저장 및 분석 기반 구축

로그 중심 구조에서 DB 기반 관리 구조로 확장하는 단계입니다.

* 거래 로그 DB 저장
* 주문 이력 저장
* 체결 이력 저장
* 잔고 이력 저장
* 전략별 성과 분석
* 일별 손익 분석
* 주별 손익 분석
* 월별 손익 분석
* 실패한 매수 후보 분석
* 설정값 변경 이력 관리

---

## v2.x - 웹 대시보드

자동매매 상태와 결과를 웹에서 확인할 수 있도록 대시보드를 개발하는 단계입니다.

* Vue 기반 웹 대시보드 개발
* 현재 봇 상태 표시
* 현재 운용 모드 표시
* 보유 종목 표시
* 평가손익 표시
* 주문 내역 표시
* 체결 내역 표시
* 일일 리포트 조회
* 전략별 성과 조회
* WebSocket 기반 실시간 이벤트 표시
* Replay 모드를 통한 퇴근 후 대시보드 테스트
* Mock 모드를 통한 프론트엔드 UI 테스트

---

## v3.x - 확장 기능 검토

국내장 자동매매 안정화 이후 검토할 확장 기능입니다.

* 미국장(NASDAQ/NYSE) 데이터 조회 검토
* 미국장 Read-only 대시보드 검토
* 미국장 Paper Trading 검토
* 미국장 실거래 지원 여부 검토
* 미국장 소수점 자동매매 지원 여부 검토
* 전략별 백테스트 고도화
* 알림 기능 검토
* 모바일 대응 검토

현재 버전에서는 미국장 실거래 및 미국장 소수점 자동매매를 지원하지 않습니다.
